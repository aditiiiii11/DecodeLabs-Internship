# DecodeLabs UI/UX Internship Projects

This repository contains the projects completed during the DecodeLabs UI/UX Design Industrial Training Program.

## Week 1 – The Empathy Map

### Problem Statement

A fitness startup wanted to launch an app for busy office professionals who struggle to find time for exercise.

### Objectives

* Define the target audience
* Create User Personas
* Build an Empathy Map
* Understand user needs, goals, and frustrations

### Deliverables

* User Persona
* Empathy Map

### Tools Used

* Canva

---

## Week 2 – The Blueprint (Wireframing)

### Problem Statement

An e-commerce platform was experiencing customer drop-offs due to a cluttered layout and confusing navigation.

### Objectives

* Create a low-fidelity homepage wireframe
* Improve information hierarchy
* Design a clear user flow
* Add effective Call-to-Action (CTA) buttons

### Deliverables

* E-commerce Homepage Wireframe
